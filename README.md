Personal Portfolio Website 

This is my first-year portfolio website project built using HTML and CSS. The project showcases my profile, skills, projects, and contact information in a clean and responsive design.

Features
- Responsive portfolio layout
- About Me section
- Skills showcase
- Project gallery
- Contact section
- Simple and modern UI

🛠️ Technologies Used
- HTML5
- CSS3
 🎯 Purpose of the Project
The purpose of this project was to improve my frontend development skills and learn how to design and structure websites using HTML and CSS.

 📚 What I Learned
- Web page structuring using HTML
- Styling with CSS
- Responsive design basics
- Layout creation using Flexbox/Grid
- Building user-friendly interfaces

 🚀 Future Improvements
- Add JavaScript for interactivity
- Improve animations and transitions
- Add dark mode
- Make the portfolio fully dynamic
